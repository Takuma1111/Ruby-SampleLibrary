# Ruby-SampleLibrary
Rubyの簡単な拡張ライブラリを作成しました

# きっかけ
ライブラリを使用していて、ライブラリ内の動きや作り方など興味を持ちました。  
そこで簡単なライブラリでも良いので実際作ってみてどうなのか試してみたく、作ってみました。

# 概要
メソッドに引数を渡すことによってその引数のサイズを出力する拡張ライブラリです。

# 環境
Ruby 2.6.3
